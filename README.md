# StellarisOS
Minimal ARM Cortex-M3 Operating System

# How to compile
To compile, you'll need a unix system able to run make. (eg. Linux, WSL2, MSYS2, etc)
run 
```term
make run
```
to compile & run the OS.

Packages needed to compile:
```
qemu-system-arm
gnu-arm-none-eabi
binutils
```

# Current features
- Menu Interface
- Applications
- Kernel
